## Chess Game

Based on Pear Games_Chess - represents a chess game as a php object

#### Installing with a composer:

```sh
  ~ composer require chesscom/chess-game
  ~ composer update chesscom/chess-game
```

#### Running the test suite:

```
composer install
phpunit
```
